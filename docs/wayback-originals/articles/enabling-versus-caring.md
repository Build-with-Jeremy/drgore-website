# Enabling Versus Caring

| Enabling | Caring |
|----------|--------|
| I protect | I share |
| I control | I confront |
| I carry their feelings | I am sensitive |
| I don't listen | I listen |
| I rescue | I don't rescue |
| I fix | I let consequences naturally occur |
| I feel... tired, anxious, fearful, over-responsible | I feel... relaxed, free, aware, highly-esteemed |
| I can't trust | I can trust |
